🔥 Sincronização em Tempo Real

Com o Firebase, resultados digitados em qualquer celular aparecem instantaneamente para todos. Gratuito para este uso.

Como configurar (5 minutos):
1. Acesse console.firebase.google.com
2. Clique em Adicionar projeto → dê um nome → crie
3. No menu lateral: Build → Realtime Database → Criar banco
4. Escolha Modo de teste → Ativar
5. Clique na engrenagem (⚙️) → Configurações do projeto
6. Role até Seus aplicativos → clique em </> → registre o app
7. Copie o objeto firebaseConfig e cole abaixo
